Welcome to the Dynamic Graph Generator Tool

Follow the instructions below to use the tool effectively.

1. Graph Type

Select the type of the graph you want to generate. Options include:

  • Directed: The graph will have directed edges.
  • Undirected: The graph will have undirected edges.

2. Input Graph

Either choose an available graph or upload a custom graph. Supported formats are:

  • Matrix Market

3. Input Transforms

Select transformations to apply to the input graph. Available options are:

  • Transpose
  • Symmetrize

4. Batch Size

Specify the size of each batch update in absolute terms or as a fraction of the total edges. Example:

  • Batch Size: Absolute number of edges in each batch.
  • Batch Size Ratio: Fraction of the total edges (e.g., 0.001).

5. Edge Operations

Specify the percentage of edge insertions and deletions in each batch(a fraction between 0 and 1).

6. Update Nature

Specify the nature of the batch updates:

  • Uniform: Uniformly random updates.
  • Preferential: Preferential attachment model.
  • Exponential: Exponential distribution based update.

You can also define a custom probability distribution function for the batch updates. Ex: 0.01*exp(-0.01*x)

7. Constraints

Specify constraints to maintain the properties of the graph:

  • Min/Max Degree: Ensure the degree of nodes remains within the specified range.
  • Min/Max Diameter: Ensure the diameter of the graph stays within the specified range.
  • Min/Max SCC: Ensure the number of strongly connected components stays within the specified range.
  • Min/Max BCC: Ensure the number of bi-connected components stays within the specified range.
If all the constraints are satisfied, the graph will be highlighted in green; otherwise, it will be highlighted in red.

8. Multi-Batch Updates

Specify the number of contiguous batch updates to generate.

9. Miscellaneous

  • Seed: Seed for the random number generator (for reproducibility).

Compulsory Options

The following options are compulsory and must be provided:

  • Either Update Nature or Probability Distribution must be given.
  • Either Batch Size or Batch Size Ratio must be given.
  • Either a Custom Graph must be uploaded or a Pre-existing Graph must be selected.
